Had an issue where after updating code-server, I didn't have gcc, make or man pages anymore

First we are going to install vim as that isn't installed by default with the linuxserver code server

sudo apt install vim

Edit (or remove) the following file /etc/dpkg.cfg.d/excludes
And once you've done that come back here and enter the below command (installs gcc, man pages, make and other various needed things)

sudo apt --reinstall install coreutils man-db manpages-posix manpages-dev

Once all that is done, you will need to reinstall norminette as it errors out when updating the code-server

To start with, python is already installed as version 2.7 with default linux.
We can install multiple versions of python side-by-side.

To start with we want to install the prerequisites

sudo apt update
sudo apt install software-properties-common

Add deadsnakes PPA to your system's sources list

sudo add-apt-repository ppa:deadsnakes/ppa
sudo apt install python3.9

Some prerequisites before we start

Go to and make (if not already made) to ~/pip and type the following to download the pip install script

curl https://bootstrap.pypa.io/get-pip.py -o get-pip.py

As we are using python 3.9 we want to install distutils to help python

sudo apt install python3.9-distutils

Now we can install pip

sudo apt-get install python3.9-pip

If you already have a version of norminette running on your computer you will need to remove the old version as the current version may conflict with the new version that we are about to install.

pip uninstall norminette

Once we have either removed (in the case of norminette being already installed) or installed norminette we want to install pip's setuptools and finally install norminette!

python3.9 -m pip install --upgrade pip setuptools
python3.9 -m pip install norminette

After all that we check the current version of norminette (current as of time of writing)

$ norminette -v
norminette 3.3.32

A quick side-note - If we want to use any testers (relevant for libft at the moment) we also need to additionally install clang and valgrind

sudo apt install clang valgrind
