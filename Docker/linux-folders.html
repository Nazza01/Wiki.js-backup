Current View when typing commands

abc@xxxxxx:~/workspace$ <command>

Wanted like this without the whole folder directory or computer name

abc:workspace$ <command>

Fixed by adding this line in the .bashrc 

PS1='${debian_chroot:+($debian_chroot)}\[\033[01;32m\]\u:\[\033[01;34m\]\W\[\033[00m\]\$ '

Then doing this command to get it to re-fresh the file

source ~/.bashrc

All good now!
