Nextcloud needs a command to scan the directory when you modify it using a different program (code-server)

As such, you'll need to do this command (when not root)

runuser --user www-data -- chown -R www-data:www-data .

If you are root (e.g. root@/www/html/data) you won't need to do runuser

chown -R www-data:www-data .
